URL Shortener

Go to Home

This microservice helps to check if a url is valid or not, if valid, it returns a JSON response with original_url and short_url, the short_url can then be used to redirect the user to the original_url.

Demo

Request

POST /api/shorturl

Response

original_url
Original URL, which is entered by the user.
short_url
A unique generated id which can be used to redirect to the original_url by visiting /api/shorturl/<short_url>

Example Output

{"original_url":"https://www.freepcamp.org/news/the-100-correct-way-to-do-css-breakpoints-88d6a5ba1862/","short_url":0}
