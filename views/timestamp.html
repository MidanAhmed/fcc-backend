Get UTC date and time.

Request

GET /api/{date}

Response

unix
A number that represents milliseconds since the midnight at the beginning of January 1, 1970, UTC (the epoch).
utc
A string representing the given date using the UTC time zone.

Example Input

  • /api/2015-12-25
  • /api/1451001600000

Example Output

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
