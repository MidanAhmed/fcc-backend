Exercise Tracker

Go to Home

A microservice that allows us to create a new user, add an exercise to the related user and get the exercise log of an user.

Demo

  • Create a new user

  • Add exercises

Request

POST /api/users

Response

_id
A unique generated id assigned to the user.
username
The username given by the user.

Example Output

{ username: "fcc_test", _id: "5fb5853f734231456ccb3b05" }

Request

POST /api/users/:_id/exercises

Response

_id
A unique generated id assigned to the user.
username
The username given by the user.
description
The description of the exercise.
date
The date given by the user. Defaults to current date if left empty.
duration
The duration of the exercise in minutes.

Example Output

{ username: "fcc_test", description: "test", duration: 60, date: "Mon Jan 01 1990", _id: "5fb5853f734231456ccb3b05" }

Request

GET /api/users/:_id/logs?[from][&to][&limit]

Response

_id
A unique generated id assigned to the user.
username
The username given by the user.
count
Number of logs returned.
log
An array containing exercises.

Example Output

{ username: "fcc_test", count: 1, _id: "5fb5853f734231456ccb3b05", log: [{ description: "test", duration: 60, date: "Mon Jan 01 1990", }] }
